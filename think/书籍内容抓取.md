                                                 <p> 江凡揶揄一笑：“我说过自己是哑巴吗？”
</p><p> 许悠然脑海中凌乱一片，怔怔道：“你、你是在假装哑巴？为什么呀？”
</p><p> 江凡背负着手，静默望向窗外的天空。
</p><p> 俄顷。
</p><p> 晴朗的天空，接连响过两声震天的惊雷，吓得许府上下都为之一惊，纷纷出门眺望。
</p><p> 许悠然也被突如其来的雷声吓得一颤，呢喃道：“怎么凭空闪惊雷，跟三年前一样。”
</p><p> 江凡面无表情。
</p><p> 这就是他为什么装哑巴的原因。
</p><p> 他体内有一颗神秘的种子，每次开口说话，都会泄露一丝种子的气息，引来天雷怒吼。
</p><p> 从小父亲就告诫他，不要开口说话。
</p><p> 十八年来，他一直谨遵父亲的叮嘱，除了他去世时。
</p><p> 那一晚，电闪雷鸣，一夜未停。
</p><p> 许悠然见他神情间流露着一抹悲伤，只当他有难言之隐，柔声道：“好了好了，我不问了。”
</p><p> “你收拾东西快走吧，王姨要是知道你一直装哑巴，又要疑神疑鬼了。”
</p><p> 江凡转过身，轻轻抓住她的柔荑，道：
</p><p> “我不走，留下来娶你，从此保护你。”
</p><p> 许悠然俊俏的容颜刹那嫣红，耳畔回荡着“保护你”三个字，没来由的心脏加速跳动。
</p><p> 她慌张的抽回手，轻嗔道：“别说大话了，你的灵根还不如我呢！”
</p><p> 江凡笑而不语。
</p><p> 父亲临终前告诉了他种子的真相。
</p><p> 那是一颗孕育着极大潜力的神物，需要十八年才能发芽。
</p><p> 发芽后，宿主将迎来脱胎换骨的变化。
</p><p> 今天，就是十八年的最后一天。
</p><p> 江凡将随着明日天地间第一缕光辉，浴火重生！
</p><p> 那时，世上不再有哑巴江凡，更不再有废物江凡！
</p><p> 他静静注视着许悠然，自信道：“相信我。”
</p><p> 许悠然对上江凡平静而坚定的目光，心底涌入一道暖流。
</p><p> 自从母亲去世，还是第一次有人说要保护她。
</p><p> 思绪良久，她再次将钱袋塞进江凡手里。
</p><p> 江凡微微皱眉，她还是不相信自己啊。
</p><p> 也难怪，他现在还是哑巴，还是没有灵根的凡人呢。
</p><p> 等明天种子发芽……
</p><p> 许悠然注视着他，脸颊带着一丝羞红，轻声道：“拿着当娶我的聘礼吧。”
</p><p> 轰——
</p><p> 不是天空闪过惊雷。
</p><p> 是江凡的心底。
</p><p> 他从未想过，有一天，会有一个女孩拿着自己的毕生积蓄，让他当做聘礼迎娶她。
</p><p> 一股说不明道不清的情绪，缠绕着江凡的心灵。
</p><p> 让他有种深深抱住许悠然，好好保护她的冲动。
</p><p> “愣着干什么呀？”
</p><p> 许悠然伸出小手，轻轻整理他的衣服，温柔道：“江叔叔一生清贫，没给你留什么金银，这些积蓄希望能帮到你。”
</p><p> 她自己都过得艰难，心中却还在为江凡着想。
</p><p> 江凡再也抑制不住，一把搂住她的纤腰，将她拥入怀里。
</p><p> 再度嗓音柔和的开口：“也许我们的结合方式是个意外，但结局却是正确的。”
</p><p> 最美好的婚姻，不过遇上对的人。
</p><p> 也许，江凡就遇上了最对的那个人。
</p><p> 许悠然羞得满脸通红，像深秋里的柿子，大脑陷入一片空白，身体也僵硬得一动不能动，任由江凡搂着。
</p><p> 良久她才反应过来，惊慌的挣脱出来，道：“我、我量衣服去了……”
</p><p> 望着她逃也似的背影，江凡把钱袋放回桌上，微笑道：“迎娶我的妻子，怎能如此寒酸。”
</p><p> “悠然，我会风风光光娶你的。”
</p><p> 轰轰——
</p><p> 接连两道天雷炸响。
</p><p> 江凡狠狠瞪了眼天空，哼道：“炸吧炸吧，过了今晚，你们再也没有机会了！”
</p><p> 后院。
</p><p> 王映凤卧房。
</p><p> “姨母，你怎么能把悠然表妹嫁给江凡这个废物？”
</p><p> 同样寄居在许府十年的陆争，跪在王映凤面前，脸上写满了焦急：“请你收回成命，把表妹嫁给我吧！”
</p><p> 陆争是王映凤的侄儿，聪明伶俐，又是三品灵根的天才。
</p><p> 因此王映凤自小把他接到许府栽培。
</p><p> 他也很成气，才二十岁就有练气五层修为，让王映凤在娘家人面前狠狠涨了一把脸。
</p><p> 唯一让王映凤头疼的是，陆争很喜欢许悠然，并一直将她视作自己的女人。
</p><p> 为了不让他捣乱，今天还特意将他支开，让他去外地讨要一笔债务。
